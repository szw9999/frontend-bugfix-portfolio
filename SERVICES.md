# Service Packages

## Basic: Small Bug Fix

Price target: 25 USD

Delivery target: 2 days

Scope:

- One small frontend bug
- One TypeScript error
- One UI behavior issue
- One clear GitHub issue with a small fix

Deliverables:

- Code change or patch
- Short explanation
- Verification steps

## Standard: Multi-File Bug Fix

Price target: 75 USD

Delivery target: 4 days

Scope:

- One medium frontend bug
- Two or three closely related small issues
- React component behavior fix
- Build or TypeScript cleanup across a small area

Deliverables:

- Code change, patch, or pull request
- Root-cause summary
- Verification steps
- Small test update when practical

## Premium: GitHub Issue to Pull Request

Price target: 150 USD

Delivery target: 7 days

Scope:

- Investigate a GitHub issue
- Reproduce or reason through the problem
- Implement a focused fix
- Prepare a clean pull request or patch

Deliverables:

- Pull request or patch
- Root-cause summary
- Verification notes
- Test update when feasible
- PR description
