# Frontend Bug Fix Portfolio

Focused React, TypeScript, JavaScript, and GitHub issue fixes.

## Services

- React component bug fixes
- TypeScript error fixes
- JavaScript runtime error fixes
- UI behavior fixes
- Next.js frontend fixes
- GitHub issue investigation
- Pull request cleanup
- Small test updates for focused fixes

## How I Work

1. Reproduce or understand the issue from the report.
2. Keep the change focused and avoid unrelated refactors.
3. Update code with a clear implementation path.
4. Add or update a small test when the project setup makes that practical.
5. Provide a concise summary and verification notes.

## Good Fit

This service is a good fit for:

- A clear frontend bug
- A failing TypeScript or JavaScript build
- A small GitHub issue that needs a focused pull request
- A React component with incorrect UI behavior
- A codebase that already has setup instructions

## Not a Good Fit

This service is not a good fit for:

- Full app builds from scratch
- Large rewrites
- Tasks requiring personal passwords or private credentials
- Unauthorized security testing
- Scraping or automation that violates platform rules
- Work that depends on private production systems without safe test access

## Client Checklist

Please provide:

- Repository link or project files
- Bug description
- Expected behavior
- Actual behavior
- Error message, screenshot, or screen recording
- Steps to reproduce
- Setup instructions
- Preferred delivery format: pull request, patch, or changed files
